# サンプル画像の準備が必要です

このディレクトリには `sample-card.png` （名刺画像のサンプル）を配置してください。

Chapter 5（Prompt API 画像 — 名刺OCR）で、参加者が「サンプル画像を使う」ボタンを押した際に読み込まれます。

## 作成方法（いずれか）

### 方法 A: Canva 等で自作（推奨）

1. [Canva](https://www.canva.com/) や [Figma](https://www.figma.com/) で名刺サイズのデザインを作成
2. 以下の情報を含める（架空でOK）：
   - 氏名（例: 山田太郎）
   - 会社名（例: サンプル株式会社）
   - 役職（例: シニアエンジニア）
   - メールアドレス（例: yamada@example.com）
   - 電話番号（例: 03-1234-5678）
   - 住所（例: 東京都渋谷区...）
3. PNG 形式でエクスポート
4. このディレクトリに `sample-card.png` として保存

### 方法 B: パブリックドメイン素材を使う

[Pexels](https://www.pexels.com/) や [Unsplash](https://unsplash.com/) で「business card」と検索し、CC0 / Public Domain の画像を選ぶ。

### 方法 C: 開発中は手元の名刺の写真を使う

会場テスト用にリアルな名刺画像を確認したい場合、手元の名刺をスマホで撮影して一時的に配置。**公開リポジトリには絶対にコミットしない**でください。

## 推奨仕様

- 形式: PNG または JPG（PNG 推奨）
- 解像度: 幅 800px 以上（読み取り精度のため）
- ファイルサイズ: 500KB 以下（ページ読込速度のため）
- 著作権: 自作または CC0 / Public Domain

## セキュリティ注意

- 本物の個人情報を含む名刺画像はコミットしないでください
- 公開リポジトリに置くのは架空の情報のみで
