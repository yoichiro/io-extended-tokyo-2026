🏁 導入・環境チェック

このミニラボでは、Chrome Built-in AI の 4 つの API を 60 分で体験します。

📖 共通の 3 ステップパターン
  1. Api.availability() — 使えるか判定（4 状態）
  2. Api.create({...}) — セッション/インスタンス作成
  3. 実行メソッド（detect / translate / summarize / prompt）

🔤 Language Detector

🌐 Translator

📇 メタ情報
所要時間: 10分 / 難易度: ⭐️⭐️ / 使う共通ステップ: ①②③
ハイライト: downloadprogress イベントでモデル DL を可視化

🎯 ゴール：source/target 言語を選んで翻訳し、モデルダウンロード進捗を表示する。

🚨 見本コード 
const monitor = (m) => {
  m.addEventListener('downloadprogress', (e) => {
    showProgress('chapter-2', e.loaded * 100);
  });
};

const translator = await Translator.create({
  sourceLanguage,
  targetLanguage,
  monitor,
});

const translated = await translator.translate(text);
renderOutput('chapter-2', translated);

📝 Summarizer

📇 メタ情報
所要時間: 10分 / 難易度: ⭐️⭐️ / 使う共通ステップ: ①②③
ハイライト: summarizeStreaming() でリアルタイム表示

🎯 ゴール：長文を要約し、type/length オプションと streaming 出力を体験する。

🚨 見本コード 
const summarizer = await Summarizer.create({ type, length });

if (useStreaming) {
  const stream = summarizer.summarizeStreaming(text);
  let acc = '';
  for await (const chunk of stream) {
    acc += chunk;
    renderOutput('chapter-3', acc);
  }
} else {
  const result = await summarizer.summarize(text);
  renderOutput('chapter-3', result);
}

💬 Prompt API（テキスト）— 感情分析＋タグ抽出

📇 メタ情報
所要時間: 13分 / 難易度: ⭐️⭐️⭐️ / 使う共通ステップ: ①②③
ハイライト: responseConstraint で確実に JSON を返させる

🎯 ゴール：レビュー文を入力すると、{ sentiment, score, tags, summary } を JSON で返す。

🚨 見本コード 
const session = await LanguageModel.create({
  initialPrompts: [{ role: 'system', content: systemPrompt }],
  temperature,
  topK: 3,
});

const output = await session.prompt(reviewText, {
  responseConstraint: SENTIMENT_SCHEMA,
});

renderOutput('chapter-4', JSON.parse(output));

📷 Prompt API（画像）— 名刺OCR

📇 メタ情報
所要時間: 10分 / 難易度: ⭐️⭐️⭐️ / 使う共通ステップ: ①②③
ハイライト: expectedInputs: [{ type: 'image' }] のマルチモーダル

🎯 ゴール：名刺画像から連絡先情報を JSON で抽出する。

画像を選ぶとここにプレビューが表示されます
🚨 見本コード 
const session = await LanguageModel.create({
  expectedInputs: [{ type: 'image' }],
});

const output = await session.prompt([{
  role: 'user',
  content: [
    { type: 'image', value: imageBlob },
    { type: 'text', value: 'この名刺から連絡先情報を JSON で抽出してください。' },
  ],
}], { responseConstraint: BUSINESS_CARD_SCHEMA });

renderOutput('chapter-5', JSON.parse(output));
